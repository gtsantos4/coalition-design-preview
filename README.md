# Coalition Design Preview

Visual design preview for a Charlottesville mutual-aid coalition of four organizations:
United Way of Greater Charlottesville, ReadyKids, Starr Hill Pathways, and City of Promise.

Built from the Lovable DESIGN.md aesthetic (warm cream canvas, humanist text-led layout)
with a persimmon accent color and Newsreader serif headlines.

Live preview: https://gtsantos4.github.io/coalition-design-preview/
